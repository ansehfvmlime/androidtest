**기능 명세서: 탭 전환 및 AJAX 뷰어**

**목적**
- 대시보드에서 탭 클릭 시 JSON 데이터를 비동기로 로드하고 보기 좋은 카드/트리 형태로 렌더링한다.

**주요 기능**
- 탭 전환 시 해당 패널만 표시
- 최초 로드 탭은 서버 렌더링 사용(`data-preloaded="true"`)
- 미로드 탭은 AJAX로 JSON 호출 후 렌더링
- 아크그리드/아크패시브는 전용 레이아웃 렌더링
- 새로고침 버튼으로 캐시를 비우고 페이지 리로드

**처리 흐름**
1. 탭 클릭
2. `data-preloaded`면 요청 생략
3. `fetch`로 JSON 요청
4. `renderJsonPane` 또는 `renderArkPane`로 HTML 생성
5. 패널에 삽입, 로드 완료 플래그 저장

**렌더링 규칙**
- 일반 JSON: 객체/배열 트리 뷰로 표시
- 아크그리드/아크패시브:
  - 상단 상태 배지(가용/준비중)
  - 주요 원시 필드(문자열/숫자/불리언) 그리드
  - 대표 리스트(노드/이펙트) 카드 목록
  - 원본 JSON 접기/펼치기 제공
- 아크패시브는 섹션(진화/깨달음/도약)별 요약 렌더링 포함

**오류 처리**
- `fetch` 실패 시 기존 내용 하단에 실패 메시지를 표시
