**기능 명세서: 치명/공속 계산기 (프런트엔드)**

**목적**
- 캐릭터 스탯과 다양한 버프/각인/팔찌 값을 입력해 치명 확률과 공속/이속을 계산한다.

**초기화**
- `window.__SPEC__`의 스탯/치명 소스/속도 소스를 기본값으로 입력 필드에 주입한다.
- 기본 상수:
  - 치명 계수: `0.035777`
  - 치명 캡: `100`
  - 시너지(인당): `10`
  - 속도 버프: 서포터/만찬/음식

**입력**
- 치명 관련: 치명 스탯, 계수, 캡, 시너지 인원, 직업 각인, 아크패시브/장신구/팔찌/아드레날린 치명
- 속도 관련: 신속 스탯, 계수, 각종 버프/팔찌/삼각대/기본/아크코어/질량 증가 패널티
- 스킬/트라이포드 선택 및 가중치
- 펫 스탯 보너스(800 이상일 때 선택 가능)

**계산 로직**
- 기본 치명 = `치명 스탯 * 계수`
- 전역 보정:
  - 시너지 합산
  - 직업 각인 치명 보정(토글)
- 추가 보정:
  - 아크패시브/장신구/팔찌/아드레날린 치명(입력값)
- 치명 캡 적용: `min(합산, cap)`
- 스킬 치명:
  - 선택 스킬의 트라이포드 치명 보정치 합산
- 스킬 세트 평균 치명:
  - 체크된 스킬의 가중치 평균
- 공속/이속:
  - 신속 계수 적용 여부 토글
  - 기본/버프/팔찌/삼각대/아크코어/질량 증가 패널티 합산
  - 캡 적용
- 엔트리 공속 목표치(기본 21%)에 필요한 신속 스탯 계산

**출력**
- 현재 치명률, 스킬 치명률, 평균 치명률
- 공속/이속(버프 포함/항상)
- 직업 각인 치명피해/치명확률 증가량
- 계산식 브레이크다운 텍스트

**관련 파일**
